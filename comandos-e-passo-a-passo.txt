1. node -v
    -> para verificar a versão do node para instalações futuras

2. npm init
    -> para iniciar o projeto com a criação de um package.json

3. npm install typescript @types/node -D
    -> Instalação das dependencias do Typescript com suporte do Node

4. npx tsc --init
    -> Criação do arquivo tsconfig.json

5. Acessar: https://github.com/tsconfig/bases
    -> Para modificação do arquivo tsconfig.ts

6. npm install fastify @fastify/cors fastify-type-provider-zod zod
    -> Instalação do Fastify e algumas dependencias específicas

7. npm install dotenv
    -> Instalação da biblioteca Dotenv

8. npm install @biomejs/biome -D -> Dependencia de Desenvolvimento
    -> Instalação da biblioteca Biome como dependencia de desenvolvimento
   
9. npx ultracite init
    -> Instalação do Ultracite, preset do biome, através do Node Package Executor

10. docker -v
    -> Verificação da versão do docker

11. docker compose up -d
    -> Para containerizar o projeto em modo "--detach"

12. docker run hello-world
    -> Teste para verificar se o docker está funcionando no projeto
